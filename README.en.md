## Installation

### **You can use curl to download this script**

```shell
    apt install -y curl
    bash -c "$(curl -LfsS 'https://gitee.com/mo2/linux/raw/master/debian.sh')"
```

### **or wget**

```shell
    apt update
    apt install -y wget
    bash -c "$(wget -qO- 'https://gitee.com/mo2/linux/raw/master/debian.sh')"
```
